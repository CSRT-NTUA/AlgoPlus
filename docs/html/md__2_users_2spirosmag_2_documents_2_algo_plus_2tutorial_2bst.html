Mini Tutorial for the BST class

bst<T> -- creates an avl tree.

avl tree contains:

  • insert
  • remove
  • clear
  • search
  • inorder
  • preorder
  • postorder
  • visualize

insert:

#include <bst.h>
bst<int> b;
b.insert(10);
b.insert(5);
b.insert(4);
b.insert(13);
//creates a tree with elements {4,5,10,13};
