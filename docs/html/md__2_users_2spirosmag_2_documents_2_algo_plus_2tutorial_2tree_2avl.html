Mini Tutorial for the AVL tree class

avl_tree<T> -- creates an avl tree.

avl tree contains:

  • insert
  • remove
  • search
  • inorder
  • preorder
  • postorder
  • visualize

insert:

#include <avl.h>
avl_tree<int> a;
a.insert(10);
a.insert(5);
a.insert(4);
a.insert(13);
//creates a tree with elements {4,5,10,13};

}
